node.js安装之windows篇

2018-12-15 node 评论
配置环境变量  
"我的电脑"-> "属性"->"高级系统设置"->"高级"->"环境变量"  
系统环境变量下新建  
NODE_PATH = D:\program files\nodejs\node_global\node_modules  
由于改变了modules的默认地址，所以上面的用户变量也需要跟着改变，用户变量PATH修改为D:\program files\nodejs\node_global\  
注意：  
1.如果不修改用户变量的值，将会导致使用module的时候出现“xxx不是内部或外部命令，也不是可运行的程序或批处理的文件”。  
2.因为配置的环境变量涉及到全局，所以配置完成后需要重启计算机。
